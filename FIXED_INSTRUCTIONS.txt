================================================================================
  HOW TO USE THE BOT (UPDATED COMMANDS):
================================================================================

1. FIND THE BOT:
   Telegram -> Search -> @moskvich_moneybot
   Press START

2. SELECT CLUB (WITHOUT SLASH!):
   
   CORRECT:
   старт москвич
   
   or
   
   старт анора
   
   WRONG:
   /старт москвич  <- doesn't work!

3. ENTER DATA:

   нал
   
   Then paste:
   Д1 Жанна 2200
   Р7 Иван 700
   К3 Антон 500
   
   Then:
   готово
   
   Then:
   дата/записать

4. GET REPORT:

   прошу отчёт

================================================================================
  ALL COMMANDS (WITHOUT SLASH):
================================================================================

старт москвич              - Select club Moskvich
старт анора                - Select club Anora
нал                        - Enter NAL data
безнал                     - Enter BEZNAL data
готово                     - Finish input
дата/записать              - Save data
прошу отчёт                - Get report
список 2025-11-03          - Show records for date
исправить Д1 нал 2500 ...  - Edit record
удалить Д1 безнал ...      - Delete record
экспорт неделя москвич     - Export to Excel

================================================================================
  IMPORTANT!
================================================================================

1. DO NOT CLOSE the PowerShell window with the bot!

2. Write commands WITHOUT slash:
   CORRECT: старт москвич
   WRONG: /старт москвич

3. If bot doesn't respond:
   - Check that PowerShell window is open
   - Send: /start (this is the ONLY command WITH slash)
   - Then send: старт москвич
